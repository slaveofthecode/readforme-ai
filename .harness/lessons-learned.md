# Lessons Learned

> Registry of errors, causes, fixes, and lessons so the AI does not repeat the same mistakes in future phases.
>
> Written by the human (you) when reviewing each PR. The AI reads this at the start of every session.

## Entry Format

```markdown
## feat/NNN-<phase-name> — YYYY-MM-DD

### Error: short error title
- **Context:** when and where it happened
- **Cause:** what originated it
- **Fix:** what was done to resolve it
- **Lesson:** what the AI must do next time to avoid it
```
